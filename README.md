Um jogo interativo de adivinhação desenvolvido para praticar conceitos fundamentais de lógica de programação, manipulação do DOM e interatividade no navegador.

---

## 📋 Sobre o Projeto

O **Jogo do Número Secreto** desafia o usuário a descobrir um número gerado aleatoriamente pelo sistema dentro de um intervalo pré-definido. O jogo fornece feedback imediato (dicas se o número secreto é maior ou menor) e contabiliza as tentativas até a vitória.

Este projeto faz parte da trilha de lógica de programação da **Alura** em parceria com o programa **Oracle Next Education (ONE)**.

---

## 🚀 Funcionalidades

- **Geração Aleatória**: O sistema escolhe um número secreto diferente a cada nova partida.
- **Feedback Visual e Sonoro**: Mensagens indicam se o palpite foi alto ou baixo (com suporte a síntese de voz, dependendo da versão).
- **Contador de Tentativas**: Informa ao usuário quantas jogadas foram necessárias.
- **Reinicialização Inteligente**: Botão "Novo Jogo" que reseta o estado do sistema de forma fluida.
- **Interface Responsiva**: Adaptado para visualização em diferentes dispositivos.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estruturação semântica do conteúdo.
- **CSS3**: Estilização moderna e layout.
- **JavaScript (ES6+)**: Lógica do jogo, manipulação de elementos DOM e eventos de clique.
